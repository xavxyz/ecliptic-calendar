- 5 Chinese Seasons : `MovementsBoundaries[]`
- Equinox / Solstice : `MovementMagnitude[]`
- Sunrise / Sunset : `SunriseSunsetTimes[]` (https://sunrisesunset.io/api)
- Moon phases : `MoonPhase[]` (http://www.wdisseny.com/lluna/?lang=en)
